✔ 1. Desktop App — WAJIB di-upgrade
Kenapa?

Karena:

Sekarang perannya bukan hanya menerima data kamera + menampilkan UI.

Tetapi harus:

menggabungkan data kamera + MPU → sensor fusion

menaikkan confidence gesture

memutuskan gesture final

menangani reconnect MQTT otomatis

menyaring noise MPU

membaca paket real-time dan menjaga FPS UI

Saat ini logika fusi data belum ada, dan pipeline event belum rapih.

Upgrade yang dibutuhkan:

Algoritma sensor fusion (kamera + MPU)

Filtering (Kalman/Complementary) untuk MPU

Gesture decision layer
(kamera 50% + MPU 50% → keputusan final)

MQTT reconnect yang robust

Threading / queue untuk baca data real-time

Error handling MPU ketika data anomali

➡️ Desktop App = pusat otak → paling perlu dibangun ulang lebih rapi.

✔ 2. ESP32 Accel Library (MPU6050 class) — Sudah oke, cukup stabil

Kodenya:

bersih

simple

langsung pakai writeto / readfrom_mem

kompatibel Micropython

Yang bisa di-upgrade hanya opsional, misalnya:

Upgrade opsional:

tambahkan complementary filter

tambah scaling ke g/s (daripada integer raw)

deteksi gerakan dasar langsung dari ESP32 (swipe, shake, rotate)

Tapi tidak wajib.

Secara fungsional: bagus & efisien.
